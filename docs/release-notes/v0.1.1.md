# Release Notes - v0.1.1

**Release Date**: 2025-11-16

## Features
- Migrate user configuration from `.env` to `toml` in `~/.config/gira/credentials.toml`
- Add comprehensive test coverage for high-priority components. 
    - api
    - config
    - models formatters
- Added github action to run unit tests
- Added version sub-command
```sh
gira version
gira -v
```
