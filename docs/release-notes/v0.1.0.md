# Release Notes - v0.1.0

**Release Date**: 2025-11-15

## Overview

Initial release of Gira - a blazingly fast, read-only Jira CLI client written in Go. Gira provides a beautiful terminal user interface (TUI) for viewing and navigating your Jira active sprints, offering significantly better performance than the web interface.

## Features

### Core Functionality

- **Jira API Integration**: Complete integration with Jira REST API for fetching projects, boards, and sprint data
- **Active Sprint Viewer**: View and navigate active sprint issues in a clean, organized interface
- **Project & Board Navigation**: Browse through available Jira projects and boards
- **Task Board View**: See sprint tasks organized by status columns (To Do, In Progress, Done, etc.)
- **Task Details View**: Display comprehensive information about individual tasks including:
  - Task summary and description
  - Status and priority
  - Assignee information
  - Components and versions
  - Sprint details

### User Interface

- **Beautiful TUI**: Built with Bubble Tea framework using the Model-View-Update pattern
- **Responsive Layout**: Clean, terminal-based interface optimized for speed
- **Theme Support**: Consistent styling using Lipgloss for terminal rendering
- **Developer Tabs**: Tasks organized by developer for easy team overview
- **Keyboard Navigation**: Full keyboard-based navigation for efficiency
- **Footer Keybindings**: Context-aware keybinding hints displayed in footer

### Technical Features

- **Clean Architecture**: Separated concerns with distinct API, models, and UI layers
- **Mock Client Support**: Mock API client implementation for testing and development
- **Environment Configuration**: Secure credential management via `.env` file
- **Logging System**: Comprehensive logging stored in `~/.config/gira/logs/`
- **Error Handling**: Proper error state handling and user feedback
- **Debug Mode**: Optional debug mode via environment configuration

### Developer Experience

- **Go 1.24.6+**: Built with modern Go for excellent performance
- **Modular Structure**: Well-organized package structure for maintainability
- **Documentation**: Comprehensive README with setup and usage instructions
- **MIT License**: Open source under MIT license

## Installation

```bash
git clone https://github.com/mabd-dev/gira.git
cd gira
go mod download
go build -o gira
```

## Configuration

Create a `.env` file with your Jira credentials:

```env
email=your.email@example.com
secret=your_jira_api_token
domain=your_jira_domain
```

## Usage

```bash
./gira
```

## Known Limitations

As this is an initial release, the following limitations apply:

- **Read-Only Access**: No ability to create, update, or delete issues
- **Active Sprint Only**: Only displays the currently active sprint
- **No Caching**: No offline mode or data caching
